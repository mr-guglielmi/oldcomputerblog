The First Apple Laptop

By: Jonah Brown

Introduction

With a price of $7300 (Around $19000 today), the Macintosh Portable, released in 1989, was Appple’s first ever portable computer. Described as their first “luggable” computer, it had a built in battery and screen, and the computer was very heavy, bulky and expensive making it not a successful computer. Although it was not a success, it provided a foundation for all of the computers that would define Apple in later years. The computer was also primarily used as a personal computer on the go, and a powerfull and easy personal computer, even though it was clunky and heavy.

The computors specifics

The Macintosh Portable used a Moterolla 68000 that could run at 16MHz and had one ram slot that could have a maximum of 8 megabytes. For storage, the computer ran off of 1.44 megabyte floppy disks that could be inserted into the computer through a drive to be accessed. The Macintosh Partable ran MacOS as its default operating system and could run macOS 6.0.5, up to 7.5.5, which can support any software from those updates, and used the 68000 programming language and pascal primarily.

Comparison to modern computors

Being the first Apple laptop, how well does it compare to a modern laptop? Well in terms of general design, not much changed. A modern computer has a screen that opens up to show a keyboard and then instead of a trackball, modern computers have a trackpad instead. Modern computers also open from the back instead of the middle, because they have adapted to use more flat components to fit under the keyboard, so it can have a more flat design. In terms of hardware, newer macbooks use the apple m4 chip, which can run up to 60MHz, which is 275% faster than the Macintosh portable. The modern Apple laptops also have internal storage instead of floppy disks, which can have up to 1 terabyte of storage, which is more than 50000% more storage then one floppy disk, but you can have multiple floppy disks. The modern Apple laptops can also run the newest MacOS, which is MacOS version 26.
